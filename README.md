## Getting Started

Install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To create a optimized production Build run:

```bash
npm run build
npm run start
```
